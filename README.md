# SE PROJECT 2025 TEAM-13

## Project Title: AI Agent for Academic Guidance

### Overview
This project focuses on developing an AI-driven agent to enhance the learning experience for students enrolled in the IITM BS degree program. The agent serves as a virtual guide, helping students navigate course materials, adopt effective study habits, and maintain academic integrity. It nudges students toward better learning strategies, providing contextually relevant resources without offering direct answers.

---

## Tech Stack Used

- **Backend:** FastAPI (Python)
- **Frontend (UI):** React (TypeScript)
- **Testing:** Pytest (for unit and API testing)
- **Dependency Management:** Poetry (for backend dependencies)

---

## Features
- Personalized course-specific guidance.
- Dynamic support for a variety of courses, including programming.
- Resource-based suggestions using credible academic materials.
- Promotes collaborative and ethical learning practices.

---

## Setup Guide

### Backend

1. **Install Python**  
   Ensure Python version **>= 3.11** is installed on your system.

2. **Navigate to Backend Directory**  
   ```bash
   cd backend
   ```

3. **Install Poetry**  
   Poetry is used to manage backend dependencies. Install it using:  
   ```bash
   pip install poetry
   ```


4. **Navigate to the Platform Directory**  
   ```bash
   cd backend/platform
   ```
5. **Install Dependencies**  
   Install all the dependencies using:  
   ```bash
   poetry install
   ```

6. **Run the Application**  
   Use the following command to start the backend:  
   ```bash
   poetry run python -m ai_platform
   ```

## Testing

### Backend Tests
